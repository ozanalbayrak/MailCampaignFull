# MailCampaignFull

Prerequisites: 
* Maven
* Angular Client

* git clone https://github.com/ozanalbayrak/MailCampaignFull.git

Backend:
* cd MailCampaignFull/MailCampaign
* mvn spring-boot:run

Frontend
* cd MailCampaignFull/MailCampaignUI/mailcampaignui 
* npm install
* ng serve
* goto "http://localhost:4200/" in your browser
